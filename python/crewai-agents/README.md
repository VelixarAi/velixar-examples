# CrewAI + Velixar

Multi-agent system where agents share persistent memory. The researcher stores findings, the writer recalls them.

## Setup

```bash
pip install velixar crewai
export VELIXAR_API_KEY="vlx_your_key_here"
export OPENAI_API_KEY="sk-your_key_here"
```

## Run

```bash
python main.py
```
