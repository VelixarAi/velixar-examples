# Quickstart

Store and search memories in 10 lines of Python.

## Setup

```bash
pip install velixar
export VELIXAR_API_KEY="vlx_your_key_here"
```

## Run

```bash
python main.py
```

## What it does

1. Stores 3 memories (a preference, a project fact, a decision)
2. Searches semantically — "what programming language?" finds "prefers Python" even without exact match
3. Returns results ranked by relevance score
